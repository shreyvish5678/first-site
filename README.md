# My first Website
This is my first ever website

Directions:
- Clone the repo, go into your command line, terminal or bash and enter: `git clone https://github.com/shreyvish5678/first-site/`
- Install all requirements by entering: `pip install -r requirements.txt`
- Run the flask server by entering:  `python app.py`
- Navigate to the server by entering `http://127.0.0.1:5000/` into your browser
- Once it loads up, enjoy!
